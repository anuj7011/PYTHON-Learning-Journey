# Tuples in Python

## 📌 Overview

A Tuple is a collection of multiple values stored in a single variable. Tuples are ordered and immutable (unchangeable) data structures in Python.

Tuples are used when data should remain constant and should not be modified after creation.

---

## 🛠️ Important Tuple Methods

| Method  | Purpose                     |
| ------- | --------------------------- |
| count() | Counts occurrences of value |
| index() | Returns index position      |

---

## 🔄 Tuple vs List

| Feature    | Tuple   | List   |
| ---------- | ------- | ------ |
| Changeable | No      | Yes    |
| Syntax     | ()      | []     |
| Speed      | Faster  | Slower |
| Methods    | Limited | More   |

---

## 💡 Best Practices

* Use tuples for fixed data.
* Use meaningful variable names.
* Prefer tuples for constant values.
* Remember single element tuple needs comma.

---

## ❌ Common Mistakes

* Trying to update tuple values.
* Forgetting comma in single element tuple.
* Confusing tuple and list syntax.
