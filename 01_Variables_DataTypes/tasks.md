# Variables & Data Types Practice Tasks

## Beginner

1. Create a variable to store your name.
2. Create a variable to store your age.
3. Create a float variable.
4. Create a boolean variable.
5. Print all variables.
6. Check the data type of each variable.
7. Assign multiple variables in one line.
8. Assign one value to three variables.
9. Convert a string `"100"` into an integer.
10. Convert an integer into a float.

## Intermediate

11. Take your name as input.
12. Take your age as input.
13. Print both values.
14. Convert age to integer.
15. Calculate age after 10 years.
16. Store your favorite programming language.
17. Store your city name.
18. Create a constant using uppercase naming.
19. Print all stored information.
20. Display the type of every variable.

## Challenge

Create a Student Profile Program using variables.

The program should store:

* Name
* Age
* Course
* College
* City

Finally, print the complete profile in a readable format.
