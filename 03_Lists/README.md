# Lists in Python

## 📌 Overview

A List is a collection of multiple values stored in a single variable. Lists are ordered, mutable (changeable), and allow duplicate values.

Lists are one of the most commonly used data structures in Python because they can store multiple types of data together.

## 🛠️ Important List Methods

| Method    | Purpose                           |
| --------- | --------------------------------- |
| append()  | Adds element at end               |
| insert()  | Adds element at specific position |
| extend()  | Adds multiple elements            |
| remove()  | Removes specific value            |
| pop()     | Removes element by index          |
| clear()   | Removes all elements              |
| index()   | Finds index position              |
| count()   | Counts occurrences                |
| sort()    | Sorts list                        |
| reverse() | Reverses list                     |
| copy()    | Creates copy                      |

---

## 💡 Best Practices

* Use meaningful list names.
* Avoid unnecessary nested lists.
* Use list comprehension for cleaner code.
* Remember list indexing starts from 0.

---

## ❌ Common Mistakes

* Accessing an index that does not exist.
* Modifying a list while looping incorrectly.
* Confusing list and tuple.
* Forgetting that lists are mutable.
