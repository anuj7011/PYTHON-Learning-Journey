# Conditional Expression in Python

## 📌 Overview

Conditional expressions are used to make decisions in Python programs. They allow the program to execute different blocks of code based on whether a condition is True or False.

Python uses `if`, `elif`, and `else` statements for decision-making.

## 🔹 Conditional Statement Syntax

### if Statement

```python
if condition:
    statement
```

### if-else Statement

```python
if condition:
    statement
else:
    statement
```

### if-elif-else Statement

```python
if condition:
    statement
elif condition:
    statement
else:
    statement
```

---

## 🔄 Conditional Expression (Ternary Operator)

Short way to write if-else:

```python
result = "Pass" if marks >= 40 else "Fail"
```

---

## 💡 Best Practices

* Keep conditions simple.
* Use meaningful variable names.
* Avoid unnecessary nested conditions.
* Use elif instead of multiple if statements.

---

## ❌ Common Mistakes

* Forgetting colon `:`
* Wrong indentation.
* Using assignment `=` instead of comparison `==`.
* Writing complex conditions.
