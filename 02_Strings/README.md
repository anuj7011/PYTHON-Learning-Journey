# Strings in Python

## 📌 Overview

A String is a sequence of characters enclosed inside single quotes (`' '`), double quotes (`" "`) or triple quotes (`''' '''`).

Strings are one of the most commonly used data types in Python. They are used to store text data like names, messages, addresses, and other information.

## 🛠️ Important String Methods

| Method       | Purpose                                     |
| ------------ | ------------------------------------------- |
| upper()      | Converts string to uppercase                |
| lower()      | Converts string to lowercase                |
| title()      | Converts first letter of words to uppercase |
| strip()      | Removes extra spaces                        |
| replace()    | Replaces characters                         |
| split()      | Converts string into list                   |
| join()       | Joins elements                              |
| find()       | Finds position                              |
| count()      | Counts occurrences                          |
| startswith() | Checks starting character                   |
| endswith()   | Checks ending character                     |

## 💡 Best Practices

* Use meaningful string variables.
* Use f-string for formatting.
* Avoid unnecessary string modifications.
* Remember strings are immutable.

---

## ❌ Common Mistakes

* Trying to modify a string directly.
* Forgetting quotation marks.
* Mixing string and integer without conversion.
* Ignoring extra spaces.
