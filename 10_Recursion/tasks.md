# Recursion Practice Tasks

## Beginner Tasks

1. Print numbers from 1 to N using recursion.
2. Print numbers from N to 1.
3. Find factorial.
4. Find sum of natural numbers.
5. Print Fibonacci series.

## Intermediate Tasks
