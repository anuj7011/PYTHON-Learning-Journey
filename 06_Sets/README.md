# Sets in Python

## 📌 Overview

A Set is a collection of unique elements stored in a single variable. Sets are **unordered** and **mutable** data structures in Python.

Sets are mainly used when we need to store unique values and perform mathematical operations like union, intersection, and difference.

## 🔹 Set Characteristics

* Unordered ✅
* Mutable (Changeable) ✅
* Does not allow duplicate values ✅
* Does not support indexing ✅
* Allows multiple data types ✅

---

## 🛠️ Important Set Methods

| Method                 | Purpose                              |
| ---------------------- | ------------------------------------ |
| add()                  | Adds an element                      |
| update()               | Adds multiple elements               |
| remove()               | Removes element (error if not found) |
| discard()              | Removes element (no error)           |
| pop()                  | Removes random element               |
| clear()                | Removes all elements                 |
| union()                | Combines two sets                    |
| intersection()         | Common elements                      |
| difference()           | Different elements                   |
| symmetric_difference() | Unique elements from both sets       |

---

## 🔄 Set Operations

### Union

Combines all unique elements from two sets.

### Intersection

Returns common elements from two sets.

### Difference

Returns elements present in first set but not second.

### Symmetric Difference

Returns elements that are present in either set but not both.

---

## 💡 Best Practices

* Use sets when unique values are required.
* Avoid using sets when order matters.
* Use set operations for data comparison.
* Use meaningful variable names.

---

## ❌ Common Mistakes

* Trying to access set elements using index.
* Expecting output order to remain fixed.
* Adding duplicate values.
* Confusing set with dictionary.
