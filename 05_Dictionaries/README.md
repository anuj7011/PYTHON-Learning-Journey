# Dictionaries in Python

## 📌 Overview

A Dictionary is a collection of data stored in **key-value pairs**. Dictionaries are used to store information where each value can be accessed using its unique key.

Dictionaries are **ordered (Python 3.7+)** and **mutable**, which means their values can be changed after creation.

## 🔹 Dictionary Characteristics

* Stores data in key-value pairs ✅
* Mutable (Changeable) ✅
* Keys must be unique ✅
* Keys are immutable data types ✅
* Values can be any data type ✅

---

## 🛠️ Important Dictionary Methods

| Method    | Purpose                 |
| --------- | ----------------------- |
| get()     | Access value using key  |
| keys()    | Returns all keys        |
| values()  | Returns all values      |
| items()   | Returns key-value pairs |
| update()  | Updates dictionary      |
| pop()     | Removes key-value pair  |
| popitem() | Removes last item       |
| clear()   | Removes all elements    |
| copy()    | Creates dictionary copy |

## 🔄 Dictionary Example

```python
student = {
    "name": "Anuj",
    "age": 20,
    "course": "Python"
}
```

---

## 💡 Best Practices

* Use meaningful keys.
* Keep key names consistent.
* Avoid duplicate keys.
* Use get() when key may not exist.
* Keep dictionaries organized.

---

## ❌ Common Mistakes

* Using duplicate keys.
* Accessing missing keys directly.
* Confusing dictionary with list.
* Forgetting dictionary uses `{}` brackets.
